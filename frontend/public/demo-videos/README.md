# Demo Videos

This folder contains sample videos for users to try the 3D Posture Analytics platform.

## Instructions

Place your 3 demo videos here with these exact names:
- `demo1.mp4` - (e.g., "Squat Exercise")
- `demo2.mp4` - (e.g., "Running Form")
- `demo3.mp4` - (e.g., "Yoga Pose")

The videos should be:
- Format: MP4
- Duration: 5-15 seconds (short videos process faster)
- Size: Under 10MB each
- Content: Clear view of a person performing movement

After adding the videos, they will appear as selectable options in the upload interface.
